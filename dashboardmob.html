Dashboard

Dashboard is the main tool to configure bot. You can turn on-off Modules and Functions, edit Elements with dashboard. A lot of Discord bots have their dashboards on personal webpages, but with Megumin you can open it in the any of your server channels by using simple command.

Dashboard content

Modules - groups of commands

Emotions - "sad", "dance", "laugh", "happy", "hug", "kiss", "punch";
Entertainment - "meg", "dice", "botrps", "rps";
Moderation - "emb", "warn", "timeout", "kick", "ban".

Elements - custom data, that bot will use to execute some commands or functions

Prefix - bot's prefix on your servers, possible prefixes - ".", "!", "?", ";", "%", "+", "=", "-", ">", "~", "$";
Audit Channel - custom audit channel, that are used for sending messages about actions on the server;
Moderator Role - role for users, that are able to use commands from Moderation Module;
Language - bot's messages' language, available languages - English and Ukrainian.

Functions - automated actions

Auto Command Deletion - bot deletes command-messages;
Anti Discord Links - bot deletes Discord invitation links sent to the server;
Anti All Links - bot deletes all links sent to the server.
