Dashboard
➤ "dashboard" - bot will open dashboard. Syntax: [prefix]dashboard

This command is the main tool to edit Megumin! You can turn on/off modules, functions or edit elements.

*Info in square brackets means variable, that may be different in each Discord server
*Only users with ADMINISTRATOR permission are able to use it
*Default prefix is "."

➤ Modules - groups of commands. There are three modules:
Emotions - "sad", "dance", "laugh", "happy", "hug", "kiss", "punch";
Entertainment - "meg", "dice", "botrps", "rps", "ttt";
Moderation - "emb", "rename", "warn", "mute", "unmute", "kick", "ban".
➤ Elements - custom data, that bot will use to execute some commands or functions. There are two elements:
Prefix - bot's prefix on your servers, possible prefixes - ".", "!", "?", ";", "%", "+", "=", "-", ">", "~", "$";
Mute Role - role, which will be given to muted users.
➤ Functions - automated actions. There are four functions:
Auto Command Deletion - bot will delete messages, that triggers an actions;
Anti Curse Words - bot will delete messages, which contain curse words;
Anti Discord Links - bot will delete discord invitation links ($);
Anti All Links - bot will delete all links ($).

*Bot will not delete messages from users, who have ADMINISTRATOR permission
*Symbol ($) means, that only users with Premium are able to use this command or function

Emotions
 ➤ "sad" - bot sends random GIF with sad emotion. Syntax: [prefix]sad
➤ "dance" - bot will send random GIF with dance. Syntax: [prefix]dance
➤ "laugh" - bot will send random GIF with laugh emotion. Syntax: [prefix]laugh
➤ "happy" - bot will send random GIF with happy emotion. Syntax: [prefix]happy
➤ "hug" - bot will send random GIF with characters, that are hugging. Syntax: [prefix]hug [@mentioned_user]
➤ "kiss" - bot will send random GIF with characters, that are kissing. Syntax: [prefix]kiss [@mentioned_user]
➤ "punch" - bot will send random GIF with character. that punches another. Syntax: [prefix]punch [@mentioned_user]

Entertainment
 ➤ "meg" - bot will reply to your message. Syntax: [prefix]meg [your_message]
➤ "dice" - bot will start a dice game between you and the mentioned user (3 minutes cooldown). Syntax: [prefix]dice [@mentioned_user]
➤ "botrps" - bot will play rock-paper-scissors with you (1 minute cooldown). Syntax: [prefix]botrps
➤ "rps" - bot will start a rock-paper-scissors game between you and the mentioned user (3 minutes cooldown). Syntax: [prefix]rps [@mentioned_user]
➤ "ttt" - bot will play tic-tac-toe with you. Syntax: [prefix]ttt

Moderation
 ➤ "emb" - bot will create embed with your message and custom color. Syntax: [prefix]emb [color_hex_code] [your_message]
➤ "rename" - bot will rename mentioned user. Syntax: [prefix]rename [@mentioned_user] [new_nickname]
➤ "warn" - bot will give a warn or check amount of warns, or set your custom amount to mentioned user. Syntax: [prefix]warn [@mentioned_user] [give/check/set] [reason/nothing/amount]
➤ "mute" - bot will mute mentioned user. Syntax: [prefix]mute [@mentioned_user] [reason]
➤ "unmute" - bot will unmute mentioned user. Syntax: [prefix]unmute [@mentioned_user]
➤ "kick" - bot will kick mentioned user. Syntax: [prefix]kick [@mentioned_user] [reason]
➤ "ban" - bot will ban mentioned user. Syntax: [prefix]ban [@mentioned_user] [reason]
